# ديربي كفردريان V9 - صورة الفريق كلوجو للتطبيق

## التعديلات الجديدة
- تم اعتماد صورة الفريق بعد تحسينها كلوجو للتطبيق.
- تم إزالة عرض رموز الدخول التوضيحية من داخل التطبيق.
- تم تحسين واجهة دخول الإدارة لتكون أوضح وأرتب.
- بقيت صلاحيات:
  - المدير الرئيسي: إدارة النظام كاملًا.
  - مدير الفريق الثاني: تعديل فريقه فقط وإمكانية إخفاء تشكيلته عن المستخدمين.

## المستخدمون
- يدخلون مباشرة إلى الواجهة الداخلية.
- يشاهدون:
  - موعد المباراة القادمة
  - الخطط المعلنة
  - نتائج وخطط المباريات السابقة
  - عدد مرات الفوز والتعادل
